# Lovehuble — White

Frontend Next.js per l'e-commerce Lovehuble, design minimal stile Apple.

## Stack

- **Framework:** Next.js 15 (App Router)
- **Linguaggio:** TypeScript
- **Stile:** Tailwind CSS v4, MUI v7
- **Componenti:** Radix UI, shadcn/ui
- **Animazioni:** Motion (Framer)

## Avvio rapido

```bash
npm install
npm run dev
```

L'app sarà disponibile su [http://localhost:3000](http://localhost:3000).

## Script disponibili

| Comando | Descrizione |
|---|---|
| `npm run dev` | Avvia il server di sviluppo |
| `npm run build` | Build di produzione |
| `npm run start` | Avvia il server di produzione |

## Struttura del progetto

```
src/
├── app/          # Route e layout (App Router)
├── components/   # Componenti riutilizzabili
public/           # Asset statici (immagini, font)
guidelines/       # Linee guida di design
miro/             # Brief e documentazione di prodotto
```
